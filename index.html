CSS
  • p{color: purple;}
    • p is the sector
    • color is to change the color of the text in p
    • purple is to change p text to purple
  • link rel="stylesheet" href="basic.css" type="text/css"
    • href is the css filename
  • .myClass{font-family:Arial, Helvetica, sans-serif; color: #23FBFF;background-color: #6E3B04;}
    • .myclass is the name of the class
    • font-family is to change the font of text
    • color is to change text color
    • #23FBFF is hexadecimal to change color
    • background-color is to change the background color of myclass
  • #myElement{font-family:Arial, Helvetica, sans-serif;color:#23FBFF;background-color: #6E3B04;}
    • #myElement is the id name
    • font-family is to change the font of text
    • color is to change color of the text in myElement id
    • #23FBFF is hexadecimal to change color
    • background-color is to change the background color of myclass
  • font-size: 12px
    • font-size is to change font size
    • 12px is to change the font size of the text to 12px
